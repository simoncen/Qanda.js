Getting Started

Setting up:

Users need to include the script for both jQuery and Qanda.js before their own script in the head tag of the html file.

<head>
    <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
    <script defer type="text/javascript" src='qanda.js'></script>
    <script defer type="text/javascript" src='yourCode.js'></script>
</head>

Code Snippet:

Code snipet for some basic functionality you can write in your own javascript files.

//1.
const questions = new qandaQuestion();
//2.
questions.addQuestion('Are you satisfy with your scores?', 1, 1); 
//3.
questions.addUserResponse(1);

//4.
questions.addQuestion('Which year are you in?', 3, 4);
//5.
questions.removeOptionArr();
questions.removeOptionArr();
//6.
questions.addUserResponse(2);
//7.
questions.firstMultipleChoice(2, 'first year')
questions.secondMultipleChoice(2, 'second year');
questions.thirdMultipleChoice(2, 'third year')

1: Create the qandaQuestion object.

2: Set the first question with the input text, specific type of question, and specific styling.

3: Set the response for the users regarding the first question.

4: Set the second question with the input text, specific type of question, and specific styling.

5: Remove two multiple choice options for this response.

6: Set the response for the users regarding the second question.

7: Set each of the options in the multiple choice reponse with the input text regarding the second question.


APIs:

Object creations from the library, accessible functions using the objects, and global variables.

Objects:

//1.
const title = new qandatitle();
//2.
const questions = new qandaQuestion();
//3.
const button = new qandasubmit();

1: The title object.

2: The body(question) object.

3: The submission object.

Title Object functions:

//1.
title.listStyles();     // parameters: N/A ; return type: Array of Strings on the console
//2.
title.addTitle(surveyTitle, numberOfTheStyling);     // parameters: String, Number ; return type: void
//3.
title.getTotalTitles();     //  parameters: N/A ; return type: Number

1: List out all the styling for the title offered by Qanda.js with numbers in their front which the developers can refer to when they are using the styles on the console .

2: Create the title on the DOM by passing in a String of title text and the number referring to which styling the developer is planning to use by using the function described above.

3: Return the number of titles the developer has created.

Title Object Global Variables

Access by: title.nameOfTheVariable

//1.
titles     // return type: Array Objects
//2.
styleArrNum     // return type: Array of  Strings
//3.
styleArr     // return type: Array of Strings

All these global variables for the objected can be manipulated by the developers.

1: Refer to the number of title objects the developer has created.

2: Refer to to the options of styling the developer can choose from.

3: Refer to the title stylings provided by Qanda.js.

Body Object functions:

//1.
questions.listQuestionTypes();     // parameters: N/A ; return type: Array of Strings on the console
//2.
questions.listQuestionsStyles();     // parameters: N/A ; return type: Array of Strings on the console
//3.
questions.addQuestion(quesitonText, numberOfQuestionResponseType, numberOfTheStyling);     // parameters: String, Number, Number ; return type: String on the console
//4.
questions.deleteQuestion();     // parameters: N/A ; return type: Array of Strings on the console
//5.
questions.addUserResponse(quesitonNumber);     // parameters: Number ; return type: String on the console
//6.
questions.firstMultipleChoice(questionNumber, mcOptionText);     // parameters: Number, String ; return type: void
//7.
questions.secondMultipleChoice(questionNumber, mcOptionText);     // parameters: Number, String ; return type: void
//8.
questions.thirdMultipleChoice(questionNumber, mcOptionText);     // parameters: Number, String ; return type: void
//9.
questions.forthMultipleChoice(questionNumber, mcOptionText);     // parameters: Number, String ; return type: void
//10.
questions.fifthMultipleChoice(questionNumber, mcOptionText);     // parameters: Number, String ; return type: void
//11.
questions.getTotalQuestion();     // parameters: N/A; return type: Number
//12.
questions.addOptionArr();     // parameters: N/A; return type: String on the console
//13.
questions.removeOptionArr();     // parameters: N/A; return type: String on the console

1: List out all the response types of the question which the developers can choose from according to the numbers in the front.

2: List out all the styling of the question which the developers can choose from according to the numbers in the front.

3: Add the question which the developer defines by the input text, response type, and styling option to the DOM.

4: Delete the last question which the developer defines on the DOM.

5: Add the response to the specific question on the DOM which is indicated by the number of the question from the input.

6: Place in the text in the first option A according to the question number indicated by the developer.

7: Place in the text in the second option A according to the question number indicated by the developer.

8: Place in the text in the third option A according to the question number indicated by the developer.

9: Place in the text in the forth option A according to the question number indicated by the developer.

10: Place in the text in the fifth option A according to the question number indicated by the developer.

11: Return the total number of questions which the developer has created.

12: Increase the number of options for the multiple choice questions which the developer is going to create next(Upper limit of MC is 5).

13: Reduce the number of options for the multiple choice questions which the developer is going to create next(Lower limit of MC is 1).
