Example Survery Form Created by Qanda.js:

Title Section:

A variety of styling is given for the title of the survery. User will be able to create a unique title text with Qanda.js.

To checkout and edit the different styles, please check out the Documentation page.


Body Section:

Different question types and styling is offered for the main content of the survey. User will be able to create unique question and response text with Qanda.js.

To checkout and edit the styles and question types, please check out the Documentation page.



Submission Section:

A variety of styling is given for the submission button of the survery. User will be able to create a unique submission button with a response message appearing after the clicking event.

To checkout the different styles and edit the given styles, please check out the Documentation page.
